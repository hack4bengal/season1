## How to Contribute

* Fork this repo.
* Clone the forked repo.
* Create a new branch.
* Make changes to the code.
* Commit the changes.
* Push the changes to the forked repo.
* Create a pull request.
* Your PR will be merged soon.

    Fork[Fork the project]-->branch[Create a New Branch]
